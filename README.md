# GoMuseum Sulsel

Aplikasi berbasis Android yang dirancang untuk menyediakan informasi lengkap dan terstruktur mengenai wisata sejarah dan museum di Sulawesi Selatan.

## Deskripsi Aplikasi
Aplikasi GoMuseum Sulsel memuat berbagai kategori museum seperti museum sejarah, budaya, kerajaan, dan perjuangan. Aplikasi ini dilengkapi dengan integrasi peta (Location Based Service) untuk membantu pengguna menemukan lokasi dan rute museum dengan mudah. Aplikasi ini murni berfokus pada media informasi dan edukasi sejarah, sehingga tidak menyediakan fitur transaksi atau pembelian tiket wisata.

## Teknologi yang Digunakan
* **Platform:** Mobile (Android)
* **Bahasa Pemrograman:** Java
* **Database Utama:** Firebase Realtime Database
* **Autentikasi Pengguna:** Firebase Authentication
* **Lingkungan Pengembangan (IDE):** Android Studio
* **Metode Pengembangan:** Agile

## Fitur Utama
1. Login dan Registrasi Pengguna
2. Daftar Kategori Museum (Sejarah, Budaya, Kerajaan, Perjuangan)
3. Fitur Pencarian Destinasi
4. Halaman Detail Informasi Museum
5. Integrasi Peta (Maps) Navigasi

## Panduan Instalasi (Untuk Tim Developer)
1. Lakukan *clone repository* ini ke komputer lokal:
   ```bash
   git clone url
2. Buka aplikasi Android Studio.
3. Pilih opsi Open, lalu arahkan ke folder hasil clone tersebut.
4. Tunggu beberapa saat hingga proses sinkronisasi Gradle selesai.
5. Dapatkan file google-services.json dari administrator proyek (berhubung file ini bersifat private dan tidak diunggah ke GitHub), lalu letakkan di dalam folder app/.
6. Tekan tombol Run untuk menguji aplikasi pada emulator atau device fisik.
